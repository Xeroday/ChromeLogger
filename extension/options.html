Keylog Save Type

  • "Normal" keylogging mode records basic key presses.
  • "Detailed" keylogging mode records all keyboard buttons (incl. backspace, shift, etc...).
  • "Log System Keys" will record all system keys, such as CTRL and ALT. It is only available if you select Detailed keylogging.
